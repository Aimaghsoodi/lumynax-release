# SovereignCode Live

Public, interactive policy evaluator for **AbteeX SovereignCode**.

Paste a Data Capsule and an action request &mdash; the policy decision point returns `allow`, `deny`, or `allow_with_obligations` with full reasons and an audit record. No code is executed; this is a pure policy-evaluation surface.

- Model repo: https://huggingface.co/AbteeXAILab/sovereigncode
- Sister: [LumynaX MaramaRoute](https://huggingface.co/AbteeXAILab/marama-route)
- Brand: https://abteex.com &middot; https://lumynax.com
